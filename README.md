Analysis.bbpackage
==================

> Scripts for using BBEdit with R, Stata and Mplus.

## Scripts

- Run in Mplus.applescript
- Run in R.applescript
- Run in Stata.applescript
- RUN SCRIPT.applescript
- Tidy R code.applescript
